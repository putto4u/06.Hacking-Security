1. 아키텍처 개요 (Architecture Overview)

**Social-Engineer Toolkit (SET: 사회 공학적 공격 도구 모음)**은 인간의 심리적 취약점을 이용하는 **Social Engineering (사회 공학)** 공격을 자동화하기 위해 설계된 오픈 소스 파이썬 프레임워크입니다. 이 도구는 공격자가 복잡한 공격 인프라를 수동으로 구축하는 대신, 메뉴 기반의 인터페이스를 통해 이메일 발송 서버, 가짜 웹 사이트, 악성 페이로드 등을 신속하게 구성할 수 있도록 지원합니다.

[도식: 사회 공학 공격 워크플로우]

SET Attacker Node (Kali Linux)
Payload/URL Delivery
Target Victim (User Psychology)

* 주석: SET은 단순히 기술적 취약점(Vulnerability)을 넘어서, 사람의 신뢰(Trust)와 호기심을 공격의 핵심 벡터(Vector)로 삼습니다.

2. 실습 구성 요소 (Components)

NODE 01

공격자 시스템 (Attacker)

  • OS: Kali Linux (2024.x 이상 권장)
  • Tools: SEToolkit, Metasploit Framework(MSF)
  • Network: 외부 접속 가능한 고정 IP 또는 Local IP
NODE 02

대상 시스템 (Victim)

  • OS: Windows 10/11 (보안 취약점 테스트용)
  • Browser: Chrome, Edge, Firefox 등 최신 브라우저
  • App: Microsoft Office (FileFormat 공격 테스트용)

MENU 1

Social-Engineering Attacks (사회 공학 공격)

인간의 심리를 공략하는 가장 방대한 공격 카테고리입니다.

1.1 Spear-Phishing Attack Vectors (스피어 피싱 공격 벡터)

  • Perform a Mass Email Attack: 다수의 타겟에게 악성 파일/링크 이메일 전송.
  • Create a FileFormat Payload: PDF, Word 등 문서 파일에 악성 코드(Exploit) 삽입.
  • Create a Social-Engineering Template: 이메일 본문 디자인 생성.

1.2 Website Attack Vectors (웹사이트 공격 벡터)

Java Applet: 가짜 자바 인증을 통한 시스템 권한 획득
Metasploit Browser Exploit: 브라우저 제로데이 공격
Credential Harvester: 가짜 로그인 페이지 정보 수집
Tabnabbing/Web Jacking: 탭 교체 및 리디렉션
HTA Attack: HTML 애플리케이션 실행을 통한 파워쉘 공격

1.3 ~ 1.6 Hardware & Media Vectors

  • 1.3 Infectious Media Generator: 감염된 USB/CD 생성 (Autorun 기능 활용).
  • 1.4 Create a Payload and Listener: 실행 파일 형태의 페이로드 생성 및 리스너 대기.
  • 1.5 Mass Mailer Attack: 독립적인 대량 메일 발송 모듈.
  • 1.6 Arduino-Based Attack: 아두이노(HID) 기기를 이용한 키보드 입력 공격.

1.7 ~ 1.9 Wireless & Modern Vectors

  • 1.7 Wireless Access Point Attack: 가짜 AP를 생성하여 트래픽 스니핑.
  • 1.8 QRCode Generator: 악성 주소가 담긴 QR 코드 생성.
  • 1.9 Powershell Attack Vectors: 파일 없이 실행되는 파워쉘 기반의 리버스 쉘(Reverse Shell) 공격.

4. 아키텍트 인사이트 (Expert Insight)

실습 성공을 위한 핵심 팁

Credential Harvester나 Site Cloner를 사용할 때, 대부분의 최신 웹사이트는 **HSTS (HTTP Strict Transport Security, 강력한 HTTP 보안 전송)**가 적용되어 있습니다. 따라서 실습 시에는 `http` 기반의 사이트를 대상으로 하거나, 공격자 시스템에 SSL 인증서를 설정해야 브라우저의 경고를 우회할 수 있습니다.

법적 및 보안 주의 사항

SET은 매우 강력한 파괴력을 가진 도구입니다. **인가되지 않은 시스템에 대한 공격 시도는 정보통신망법 위반**으로 심각한 형사 처벌을 받을 수 있습니다. 반드시 외부 인터넷과 단절된 **격리 환경(Sandboxed Environment)**에서 본인 소유의 시스템으로만 실습하십시오.
