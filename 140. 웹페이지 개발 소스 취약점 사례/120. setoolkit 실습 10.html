1. 사회 공학적 공격 아키텍처 (Attack Vector Architecture)

Social-Engineer Toolkit (SET, 사회 공학 도구 세트)은 기술적 방화벽이 아닌 '사람'이라는 시스템의 가장 약한 고리를 공략합니다. 공격자는 타겟이 신뢰하는 환경을 Spoofing (스푸핑, 위장)하여 자발적인 권한 부여나 정보 입력을 유도합니다.

[도식: 사회 공학적 침투 및 권한 상승 흐름]

Deception (기만)
Phishing/Cloning
Execution (실행)
User Interaction
Access (접근)
Reverse Shell

2. 실전 공격 벡터 상세 가이드 (10 Attack Scenarios)

01

자격 증명 수집 (Credential Harvester Attack Method)

Detailed Menu Selection (상세 선택 경로)

1 1) Social-Engineering Attacks (사회 공학 공격)
2 2) Website Attack Vectors (웹사이트 공격 벡터)
3 3) Credential Harvester Attack Method (자격 증명 수집 방식)
4 2) Site Cloner (사이트 복제 도구)
5 IP Address for POST: (공격자 Kali IP 입력)
6 URL to Clone: (Metasploitable 2의 DVWA 주소 입력)

공격 해설 (Description)

사용자가 평소 신뢰하는 로그인 폼의 시각적 요소를 복제하여 경계심을 해제하는 심리적 기법을 사용합니다. 대상 서버의 로그인 페이지를 그대로 복제하여 공격자의 서버에 띄운 뒤, 사용자의 입력을 유도합니다.

기술적으로는 HTML의 <form> 태그 내 action 속성을 공격자 서버의 수집 스크립트인 post.php로 리다이렉션하여 평문 데이터를 가로채는 원리를 가집니다.

02

자바 애플릿 공격 (Java Applet Attack Method)

Detailed Menu Selection (상세 선택 경로)

1 1) Social-Engineering Attacks (사회 공학 공격)
2 2) Website Attack Vectors (웹사이트 공격 벡터)
3 1) Java Applet Attack Method (자바 애플릿 공격 방식)
4 2) Site Cloner (사이트 복제 도구)
5 Select Payload: 2) Meterpreter Reverse TCP (리버스 TCP 페이로드)

공격 해설 (Description)

시스템 업데이트나 보안 패치가 필요하다는 메시지를 통해 사용자의 '보안 준수 의지'를 자극하는 사회 공학적 유도를 수행합니다. 타겟이 'Run' 버튼을 누르는 순간 공격이 시작됩니다.

기술적으로는 Sandbox (샌드박스, 격리 영역) 외부로 실행 권한을 요청하는 서명된 자바 애플릿을 통해 운영체제의 명령줄 인터페이스에 접근하며, 최종적으로 리버스 TCP 연결을 형성합니다.

03

웹 자킹 (Web Jacking Attack Method)

Detailed Menu Selection (상세 선택 경로)

1 1) Social-Engineering Attacks (사회 공학 공격)
2 2) Website Attack Vectors (웹사이트 공격 벡터)
3 5) Web Jacking Attack Method (웹 자킹 공격 방식)
4 2) Site Cloner (사이트 복제 도구)

공격 해설 (Description)

하이퍼링크의 시각적 정보와 실제 목적지가 다를 수 있다는 점을 이용합니다. "여기를 클릭하십시오"와 같은 안내 문구 뒤에 전혀 다른 악성 주소를 숨깁니다.

기술적으로는 앵커 태그(<a>)의 상태바 텍스트를 고정하거나 onmousedown 이벤트를 조작하여 사용자가 클릭하는 순간 실제 이동 경로를 공격자의 사이트로 변경합니다.

04

탭내빙 (Tabnabbing Attack Method)

1 1) Social-Engineering Attacks (사회 공학 공격)
2 2) Website Attack Vectors (웹사이트 공격 벡터)
3 4) Tabnabbing Attack Method (탭내빙 공격 방식)
4 2) Site Cloner (사이트 복제 도구)

공격 해설 (Description)

사용자가 브라우저 탭을 여러 개 띄워놓았을 때 발생하는 주의력 분산을 이용합니다. 사용자가 다른 탭에 집중하는 사이 원래 열려있던 탭의 내용을 몰래 바꿉니다.

기술적으로는 자바스크립트의 onblur 이벤트를 감지하여 탭이 비활성화될 때 location.href를 조작, 가짜 세션 만료 페이지로 전환하여 재로그인을 유도합니다.

05

HTA 공격 (HTA Attack Vector)

1 1) Social-Engineering Attacks (사회 공학 공격)
2 2) Website Attack Vectors (웹사이트 공격 벡터)
3 8) HTA Attack Method (HTA 공격 방식)
4 2) Site Cloner (사이트 복제 도구)
5 Select Payload: 2) Meterpreter Reverse TCP

공격 해설 (Description)

업무용 문서나 필수 유틸리티 파일로 위장하여 실행을 유도합니다. 브라우저에서 파일을 '열기' 하도록 유도하는 심리전이 동반됩니다.

기술적으로는 mshta.exe라는 윈도우 기본 신뢰 실행 파일을 이용해 브라우저의 보안 제약을 우회하고 파워쉘(PowerShell)을 호출하여 메모리 상에서 악성 코드를 직접 실행합니다.

06

브라우저 익스플로잇 (Metasploit Browser Exploit)

1 1) Social-Engineering Attacks (사회 공학 공격)
2 2) Website Attack Vectors (웹사이트 공격 벡터)
3 2) Metasploit Browser Exploit Method (브라우저 취약점 공격)
4 1) I'll pick which exploit to use (수동 취약점 선택)

공격 해설 (Description)

사용자가 페이지를 방문하기만 해도 공격이 이루어지는 Drive-by Download (드라이브 바이 다운로드) 기법을 사용합니다. 별도의 클릭 유도 없이 접속만으로 충분합니다.

기술적으로는 타겟 브라우저의 버전 정보(User-Agent)를 식별한 후 해당 버전에 존재하는 메모리 변조 취약점을 자동으로 매칭하여 익스플로잇 코드를 주입합니다.

07

정보 수집 도구 (Information Gathering Tool)

1 1) Social-Engineering Attacks (사회 공학 공격)
2 2) Website Attack Vectors (웹사이트 공격 벡터)
3 10) Multi-Attack Web Vector (복합 공격 모드)
4 1) Information Gathering Tool (정보 수집 도구)

공격 해설 (Description)

본격적인 침투 전, 타겟의 기술 사양을 미리 파악하여 공격 성공률을 높이는 사전 정찰 단계입니다. "방문자 통계"나 "설문 조사" 등으로 위장합니다.

기술적으로는 Browser Fingerprinting (브라우저 핑거프린팅) 기술을 사용하여 타겟의 IP, 운영체제, 설치된 플러그인 목록을 자바스크립트로 수집하여 공격자에게 전송합니다.

08

스피어 피싱 공격 (Spear-Phishing Attack Vector)

1 1) Social-Engineering Attacks (사회 공학 공격)
2 1) Spear-Phishing Attack Vectors (스피어 피싱 벡터)
3 1) Perform a Mass Email Attack (대량 메일 공격)
4 1) Use a default template (기본 템플릿 사용)

공격 해설 (Description)

특정 개인이나 조직의 내부 정보를 인용하여 신뢰도를 극대화하는 맞춤형 속임수입니다. 지인이나 상급자를 사칭하여 의심을 최소화합니다.

기술적으로는 SMTP (Simple Mail Transfer Protocol, 간이 우편 전송 프로토콜)의 발신자 변조 취약점을 사용하여 도메인을 스푸핑하고, 악성 링크나 파일을 첨부합니다.

09

감염 매체 생성 (Infectious Media Generator)

1 1) Social-Engineering Attacks (사회 공학 공격)
2 3) Infectious Media Generator (감염 매체 생성기)
3 1) File-Format Exploits (파일 포맷 취약점 공격)
4 13) Adobe PDF Embedded Exe (PDF 내장 실행파일)

공격 해설 (Description)

'누군가 잃어버린 물건'에 대한 호기심이나 업무상 필요한 미디어로 위장하여 물리적 접촉을 유도합니다. 사무실 복도 등에 의도적으로 USB를 흘려두는 기법이 대표적입니다.

기술적으로는 윈도우의 Autorun.inf 기능이나 문서 파일의 특정 매크로 취약점을 이용하여 미디어 삽입 시 백그라운드에서 악성 바이너리가 실행되도록 설계합니다.

10

QR 코드 공격 (QR Code Generator Attack Vector)

1 1) Social-Engineering Attacks (사회 공학 공격)
2 4) Create a QR Code Generator Attack Vector (QR 코드 생성)
3 Enter the URL you want the QR code to point to:
4 (공격자의 피싱 페이지 URL 입력)

공격 해설 (Description)

물리적 공간에 부착된 QR 코드는 그 목적지를 육안으로 확인할 수 없다는 '불투명성'을 이용합니다. 식당 메뉴판이나 이벤트 전단지에 악성 코드를 덧붙입니다.

기술적으로는 짧은 URL(Shortened URL) 뒤에 피싱 사이트나 취약점 공격 코드가 담긴 주소를 숨겨 모바일 기기의 브라우저 취약점을 공략합니다.

3. 전문가적 대응 방안 (Offensive Insight)

지능형 위협 탐지 (Advanced Threat Detection)

사회 공학 공격은 기술적 패턴보다 '사용자 행위'에 집중합니다. UEBA (User and Entity Behavior Analytics, 사용자 및 엔터티 행동 분석) 솔루션을 도입하여 비정상적인 사이트에 로그인하거나 대량의 데이터를 외부로 전송하는 행위를 실시간으로 모니터링해야 합니다.

제로 트러스트 원칙 (Zero Trust Principles)

"절대 믿지 말고 항상 검증하라"는 원칙을 적용해야 합니다. MFA (Multi-Factor Authentication, 다요소 인증)는 자격 증명 탈취 공격을 무력화하는 가장 강력한 수단이며, 링크 클릭 전 도메인의 인증서와 평판을 검사하는 프로세스가 필요합니다.
