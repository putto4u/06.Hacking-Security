입력기 활성화 및 환경 설정
설치가 완료되면 GUI (Graphical User Interface, 그래픽 사용자 인터페이스) 상에서 입력기를 추가하고 단축키를 설정해야 합니다.

Terminal (터미널, 명령줄 입력창)에서 ibus-setup을 실행합니다.

Input Method (입력 방식, 입력 도구 선택) 탭으로 이동합니다.

Add (추가, 새로운 언어 엔진 등록) 버튼을 누르고 Korean -> Hangul을 선택합니다.

설정 창에서 한영 전환 키(기본값: Space + Super 등)를 확인하거나 수정합니다.

sudo dpkg-reconfigure locales
설정 창에서 ko_KR.UTF-8 UTF-8 항목을 찾아 스페이스바를 눌러 선택합니다.
기본 로케일을 ko_KR.UTF-8로 지정하고 확인을 누릅니다.
시스템을 Reboot (재부팅, 시스템 다시 시작)하면 변경 사항이 적용됩니다.
Terminal (터미널, 명령줄 입력창)에 ibus-setup 명령어를 입력하여 설정 창을 실행합니다.
