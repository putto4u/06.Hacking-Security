## Snort 2.9 실습 환경

1. victim 서버 :  v20 or v24 에서 snort2.9 apt 설치
               :  Apache + php + miriaDB 설치, https  ssl 설치 및 서비스 환경
   
2. 공격서버 : Kali 서버, snort 설치 여부 무관

3. 실습 내용
       1) snort 설치 후 기본 작동 방법
       2) 기본 Rule 설정 및 IDS 가동
       3) 외부 기본 접속 탐지 : ICMP, Http, 8080 접속, https 접속 탐지
                               특정 문자 탐지...
       4) 공격시 실시간 패킷 모니터링
       5) 공격유형 rule 만들고 탐지하기
       6)  IPS 공격 차단하기
