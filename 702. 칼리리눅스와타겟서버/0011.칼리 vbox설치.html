VirtualBox 7.x Manual

STEP-BY-STEP KALI GUIDE

최신 버전 UI 완벽 반영 & 고급 편의 설정 총망라 가이드

1. 파일 준비 및 보관

A. 공식 사이트 주소 (VirtualBox 전용)

https://www.kali.org/get-kali/#kali-virtual-machines

B. 파일 보관 규칙 (절대 준수)

다운로드하신 kali-linux-2025.4-virtualbox-amd64 폴더를 바탕화면보다는 D 드라이브나 문서 폴더 등 영구적으로 지우지 않을 장소로 먼저 옮기세요. VirtualBox 등록 후 이 폴더를 옮기거나 지우면 가상 머신이 망가집니다.

2. VirtualBox에 가상 머신 등록하기 (핵심)

정확한 메뉴 위치

1 메뉴바를 통한 파일 열기 (Open)

  1. 클릭 1. VirtualBox 상단 맨 왼쪽의 머신(M) 글씨를 클릭합니다.
  2. 클릭 2.

    펼쳐진 메뉴에서 위에서 두 번째에 있는 Open... (Ctrl+A) 항목을 클릭합니다.

    * 최신 버전에서는 '추가'가 'Open...'으로 번역되어 있습니다. '새로 만들기(N)'를 누르지 않도록 주의하세요!

  3. 클릭 3. 파일을 선택할 수 있는 탐색기 창이 나타납니다. 다운로드한 폴더로 이동하여 파란색 아이콘인 .vbox 파일을 클릭하고 하단의 열기 버튼을 누릅니다.
  4. 클릭 4. 왼쪽 목록에 'Kali Linux'가 성공적으로 등록됩니다. 목록에서 선택 후 초록색 화살표 시작(T) 버튼을 눌러 부팅합니다.

가장 직관적인 방법 (더블클릭)

메뉴를 찾기 어렵다면, 바탕화면이나 탐색기 폴더에서 kali-linux-2025.4-virtualbox-amd64.vbox 파일(파란색 네모 상자 아이콘)을 마우스로 더블 클릭하기만 해도 VirtualBox가 자동으로 열리며 등록됩니다.

3. 설정(Settings) 메뉴 상세 해설

등록된 Kali를 클릭하고 상단 메뉴의 톱니바퀴 설정(S) 버튼을 눌렀을 때의 항목들입니다.

📁 일반 (General)

[기본] 탭: 이름(Kali 등 자유 설정), 종류(Linux), 버전(Debian 64-bit)이 선택되어 있어야 합니다.
[고급] 탭: 클립보드 공유와 드래그 앤 드롭을 모두 [양방향]으로 변경하세요. (윈도우에서 복사한 글씨를 칼리에 붙여넣기 가능)

💻 시스템 (System)

[마더보드] 탭: 기본 메모리 슬라이더를 조절합니다.
* 보통 2048MB ~ 4096MB 사이로 설정하되, 초록색 영역을 넘지 마세요.
[프로세서] 탭: 프로세서 개수를 조절합니다.
* 1개면 부팅도 안 될 만큼 느립니다. 최소 2개 이상 할당을 강력 추천합니다.

🌐 네트워크 (Network) 및 어댑터 추가

[어댑터 1] 탭: 기본 네트워크

  • NAT (기본값): 내 윈도우 PC를 통해 인터넷만 연결. 외부 해킹 위협으로부터 안전.
  • 어댑터에 브리지: 공유기에 직접 연결된 컴퓨터처럼 작동. 다른 기기(공유기 대역)와 직접 통신할 때 사용.

팁: 네트워크 카드(어댑터) 2개 동시에 사용하기

실습을 하다 보면 인터넷용(NAT)과 내부망용(Host-Only 등) 랜카드가 동시에 2개 필요할 때가 있습니다. 이때 어댑터를 추가합니다.

  1. 가상 머신이 꺼져 있는 상태에서 설정 -> 네트워크로 들어갑니다.
  2. 상단의 [어댑터 2] 탭을 클릭합니다.
  3. '네트워크 어댑터 사용하기(Enable Network Adapter)' 체크박스에 체크합니다.
  4. '다음에 연결됨'에서 원하는 통신 방식(예: 호스트 전용 어댑터)을 선택하고 [확인]을 누릅니다.
  5. * 리눅스 부팅 후 터미널에 ip a를 치면 eth0 외에 eth1이라는 두 번째 랜카드가 생성된 것을 볼 수 있습니다.

❓ 디스크(.vdi) 파일을 따로 선택해야 하나요?

"머신(M)" 메뉴에서 무엇을 눌렀느냐에 따라 다릅니다!

  • ✓ 위에서 안내한 [Open... (Ctrl+A)]를 통해 .vbox를 열었다면? → 이미 .vdi 파일이 내장되어 있으니 절대 건드릴 필요 없습니다.
  • ! 만약 실수로 [새로 만들기(N)...]를 눌러 껍데기를 새로 만들고 있다면? → 설정 도중 '저장소' 단계에서 수동으로 '기존 가상 하드 디스크 파일 사용'을 선택하고 사용자님의 .vdi 파일을 찾아 연결해 주어야 합니다.

5. 네트워크 IP 마스터 (DHCP & 고정 설정)

📌 현재 IP 주소 확인하기

바탕화면이 뜨면 터미널(검은 창 아이콘)을 열고 입력하세요.

ip addr show

※ eth0 항목 아래 inet 10.0.2.15/24 처럼 적힌 부분이 내 IP입니다.

마우스 클릭(GUI)으로 DHCP / 고정 IP 변경하기 (추천)

터미널 명령어가 익숙하지 않다면 윈도우처럼 그래픽 환경에서 손쉽게 설정할 수 있습니다.

  1. 1. 연결 편집기 열기: 칼리 바탕화면 오른쪽 위 구석에 있는 네트워크 아이콘(컴퓨터 모니터 2개 모양)을 [마우스 우클릭]한 후 'Edit Connections...'를 클릭합니다.
  2. 2. 어댑터 선택: 리스트에서 변경할 네트워크(예: Wired connection 1)를 더블 클릭하거나 하단의 톱니바퀴(설정) 아이콘을 누릅니다.
  3. 3. IPv4 Settings 탭 이동: 창 상단의 [IPv4 Settings] 탭을 누릅니다.
  4. 4. 방식(Method) 선택하기:
    • 자동으로 받기 (DHCP): Method를 'Automatic (DHCP)'로 선택 후 Save.
    • 고정 IP 설정 (Static): Method를 'Manual'로 변경 → [Add] 버튼 클릭 → Address(원하는 IP), Netmask(보통 24 또는 255.255.255.0), Gateway 입력 → DNS servers에 8.8.8.8 입력 후 Save.
  5. 5. 재연결: 설정 창을 닫고, 다시 우측 상단 네트워크 아이콘을 [좌클릭]하여 'Disconnect'를 누른 후 다시 'Wired connection 1'을 눌러 재접속하면 적용됩니다.

💻 [참고] 터미널(CLI) 명령어로 고정 IP 설정하기

  1. 1단계: 터미널에 sudo mousepad /etc/network/interfaces 입력
  2. 2단계: 열린 메모장 맨 아래에 내용을 수정/추가 (본인 대역에 맞게 작성)
    # DHCP로 받을 때 (기본값)
    # auto eth0
    # iface eth0 inet dhcp

    # 고정 IP로 설정할 때
    auto eth0
    iface eth0 inet static
    address 192.168.0.50
    netmask 255.255.255.0
    gateway 192.168.0.1
  3. 3단계: 재시작: sudo systemctl restart networking

6. 실무/실습용 고급 편의 세팅

👑 최고 관리자 (Root) 계정 활성화 및 로그인

매번 명령어 앞에 `sudo`를 붙이기 귀찮다면, 최고 관리자인 `root` 계정으로 아예 로그인할 수 있습니다. 최신 칼리는 비밀번호만 부여하면 root 로그인이 열립니다.

  1. 1. 터미널 열기: 일반 `kali` 계정에서 터미널을 엽니다.
  2. 2. Root 비밀번호 설정: 아래 명령어를 치고 앞으로 사용할 root 비밀번호를 2번 입력합니다.
    sudo passwd root
  3. 3. 재로그인: 칼리 우측 상단 전원 버튼을 눌러 [Log Out] 합니다.
  4. 4. Root 로그인: 로그인 화면이 뜨면 ID란에 root, PW란에 방금 설정한 비밀번호를 치면 최고 관리자 화면으로 진입합니다!
💡 화면 꺼짐(화면 보호기/절전 모드) 완전 해제

오랜 시간 툴을 돌려놓고 기다려야 할 때 화면이 꺼지면서 잠금 화면으로 넘어가는 것을 막아줍니다.

  1. 바탕화면 좌측 상단 칼리 아이콘(🐉메뉴) 클릭
  2. Settings (설정) -> Power Manager (전원 관리자) 실행
  3. 상단의 [Display (디스플레이)] 탭을 클릭
  4. 다음 3가지 항목의 슬라이더를 맨 왼쪽으로 당겨서 'Never (안 함)'으로 만듭니다.
    • Blank after (화면 비우기)
    • Put to sleep after (절전 모드)
    • Switch off after (화면 끄기)
  5. * (추가 팁) [Security (보안)] 탭에서 'Automatically lock the session (세션 자동 잠금)' 항목도 Never로 바꾸면 잠금 화면도 뜨지 않습니다.
A. 한글 깨짐 방지 폰트

메뉴나 웹사이트의 한글을 네모 박스 없이 정상 표시합니다.

sudo apt update
sudo apt install fonts-nanum -y
B. 시스템 도구 최신화

보안 툴들의 버전을 최신으로 맞춥니다. 주기적으로 실행하세요.

sudo apt full-upgrade -y
해상도 조절 자동 맞춤 (게스트 확장 적용)

마우스로 버추얼박스 창 크기를 늘려도 내부 바탕화면이 커지지 않는다면, 실행 중인 창 상단 메뉴에서 장치(D) → 게스트 확장 CD 이미지 삽입...을 클릭하여 드라이버를 재설치하고 재부팅하세요.
